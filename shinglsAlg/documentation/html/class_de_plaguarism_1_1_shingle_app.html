virtual int DePlaguarism::ShingleApp::run ( int  port)
virtual

Runs cycle of serve.

Main client only accepts connection and puts tasks to queue. Slave objects take tasks and do all the work. Run initializes all the slaves and makes it to work. Being stopped run() waits till all the slaves finish.

Parameters
portspecifies port which will be listened by app.
Returns
0 if everything OK
